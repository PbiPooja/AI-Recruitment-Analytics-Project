# AI Recruitment Analytics | SQL Server

## Overview

Built a SQL analytics project to analyze recruitment performance and hiring outcomes using structured business data.

## Business Questions Solved

- Which recruiters achieved hiring targets?
- What is the hiring conversion rate?
- How effective is AI match scoring?
- Which candidates performed best?
- Which recruitment stages create drop-off?
